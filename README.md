# Gaming Club Landing Page

A modern, responsive landing page for a gaming club built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- Responsive design that works on mobile, tablet, and desktop
- Modern dark theme with vibrant accent colors
- Interactive components including:
  - FAQ accordion
  - Sticky call-to-action button
  - Hover animations
- Optimized for performance and SEO
- Fully typed with TypeScript

## Sections

1. Hero Section - Main value proposition and primary CTAs
2. Benefits Section - Key features of the gaming club
3. Community Hub - Discord integration and game rooms
4. Tournaments - Upcoming gaming tournaments
5. Testimonials - Member reviews and statistics
6. FAQ - Common questions and answers
7. Final CTA - Sign up form
8. Footer - Navigation and legal links

## Tech Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React](https://reactjs.org/)
